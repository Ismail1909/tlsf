# tlsf: Two-Level Segregated Fit Memory Allocator

Two-Level Segregated Fit memory allocator implementation.

## Academic Attribution

This library is an **independent C++ implementation** of the Two-Level Segregated Fit (TLSF) dynamic memory allocation algorithm described by the original authors. It is **not affiliated with, endorsed by, or maintained by** the authors of the paper or their institutions.

If this library contributes to your academic work, please cite the original publication:

> M. Masmano, I. Ripoll, A. Crespo, and J. Real,
> **"TLSF: A New Dynamic Memory Allocator for Real-Time Systems."**
> *Proceedings of the 16th Euromicro Conference on Real-Time Systems (ECRTS 2004)*,
> pp. 79–88, 2004.
> DOI: 10.1109/EMRTS.2004.1311009
